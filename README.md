# tle-eliminators

Competitive programming solutions.  
Input/output artifacts (`*.in`, `*.out`, `*.ans`) are **not** tracked.

## Structure (suggested)
- `contest/` — per-contest solutions
- `src/` — shared templates/utilities
- `testdata/` — local-only (gitignored) inputs/outputs

## Policies
- No `*.in` in Git.
- No files >100 MB.
- CI must pass on PRs.
